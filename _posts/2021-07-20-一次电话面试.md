> 很突然的一次电话面试，接到电话就问之前做过iOS，怎么做前端了。

### ES6用过哪些
let、const，这种最简单的，不存在变量提升。
高阶函数，map，filter，reduce。各个使用场景。
事后总结：
1、let、const
2、箭头函数
3、template string 模版字符串
4、Destructurin 解构
```
let a=10 ;
let obj = {
    a
}
```
5、... 运算符
6、函数的参数默认值
